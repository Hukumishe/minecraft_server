# Как отправить команды на Minecraft сервер на Railway

## Проблема

В разделе "View Logs" на Railway **нельзя отправлять команды** - это только просмотр логов. Для отправки команд нужен другой способ.

## Способы отправки команд

### Способ 1: Через игру (САМЫЙ ПРОСТОЙ) ⭐

Если вы оператор сервера:

1. **Подключитесь к серверу** в Minecraft
2. **Нажмите `/`** для открытия чата
3. **Введите команду** (например: `/save-all`)
4. **Нажмите Enter**

**Как стать оператором:**
- В Railway: **Deployments → View Logs**
- Найдите ваш никнейм Minecraft
- Или добавьте себя в `ops.json` (если Volume примонтирован к `/minecraft`)

### Способ 2: Через Railway CLI (для продвинутых)

1. **Установите Railway CLI:**
   ```bash
   npm i -g @railway/cli
   ```

2. **Войдите в Railway:**
   ```bash
   railway login
   ```

3. **Подключитесь к проекту:**
   ```bash
   railway link
   ```

4. **Подключитесь к контейнеру:**
   ```bash
   railway shell
   ```

5. **Внутри контейнера отправьте команду:**
   ```bash
   # Это сложнее, так как нужно найти процесс Java и отправить команду
   ```

**Примечание:** Этот способ сложный и не всегда работает для Minecraft сервера.

### Способ 3: Настройка RCON (для удаленного управления)

RCON позволяет отправлять команды на сервер удаленно. Но это требует дополнительной настройки.

## Рекомендация: Используйте команды в игре

**Самый простой способ** - использовать команды прямо в игре:

### Основные команды:

- `/save-all` - сохранить мир
- `/stop` - остановить сервер
- `/whitelist add <ник>` - добавить в белый список
- `/op <ник>` - сделать оператором
- `/gamemode survival` - режим выживания
- `/gamemode creative` - режим творчества
- `/time set day` - установить день
- `/weather clear` - ясная погода

### Как стать оператором:

**Вариант 1: Через команду в игре (если уже есть оператор)**
- Попросите существующего оператора выполнить: `/op ваш_ник`

**Вариант 2: Через файл ops.json (если Volume примонтирован к `/minecraft`)**
- Нужен доступ к Volume через Railway CLI или SSH

**Вариант 3: При первом запуске**
- Если вы первый игрок на сервере, добавьте себя в `ops.json` вручную

## Автоматическое сохранение

**Хорошая новость:** Minecraft **автоматически сохраняет мир** каждые несколько минут! Команда `save-all` нужна только для принудительного сохранения перед перезапуском.

## Проверка сохранения

Чтобы убедиться, что мир сохраняется:

1. Постройте что-то в игре
2. Подождите 1-2 минуты (автосохранение)
3. Или выполните `/save-all` в игре
4. Перезапустите сервер в Railway
5. Подключитесь снова - постройка должна остаться

## Итого

✅ **Используйте команды в игре** - это самый простой способ
✅ Мир сохраняется автоматически каждые несколько минут
✅ Команда `save-all` нужна только для принудительного сохранения
✅ Если Volume настроен правильно, мир сохраняется при перезапуске
