Создай современное веб-приложение для управления Minecraft сервером на Railway с темной темой в стиле Minecraft.

СТРУКТУРА ПРИЛОЖЕНИЯ:

1. Dashboard (/) - главная страница:
   - Карточка со статусом сервера (зеленый/красный/желтый индикатор)
   - Быстрые действия: Запустить/Остановить/Перезапустить
   - График использования ресурсов (CPU, память)
   - Последние события/логи
   - Навигация на все страницы

2. Загрузка сборки (/upload):
   - Input поле для URL кастомной сборки (.jar файл)
   - Валидация URL (должен вести на .jar)
   - Кнопка "Проверить ссылку" с индикатором загрузки
   - Сообщения об успехе/ошибке
   - Кнопка "Сохранить" - сохраняет в Railway переменную CUSTOM_SERVER_URL
   - Поддержка GitHub Releases, Dropbox, Google Drive и других прямых ссылок

3. Настройки мира (/world-settings):
   - Название мира (input, по умолчанию "world")
   - Seed (input, опционально)
   - Режим игры (select): Survival, Creative, Adventure, Spectator
   - Сложность (select): Peaceful, Easy, Normal, Hard
   - Toggle переключатели:
     * Разрешить PvP
     * Генерировать структуры
     * Разрешить Nether
     * Разрешить End
   - Максимальное количество игроков (number, по умолчанию 5)
   - Расстояние прорисовки (slider 4-32)
   - Расстояние симуляции (slider 4-32)
   - Кнопка "Сохранить настройки" - сохраняет в Railway переменные окружения
   - Кнопка "Сбросить к умолчаниям"

4. Управление сервером (/server):
   - Статус сервера с индикатором
   - Кнопки управления: Запустить/Остановить/Перезапустить
   - Просмотр логов в реальном времени (textarea с автообновлением)
   - Фильтрация логов по уровню (INFO, WARN, ERROR)
   - Информация о сервере:
     * Версия Minecraft
     * Игроки онлайн
     * Время работы
     * Использование памяти
   - TCP Proxy адрес с кнопкой "Копировать"
   - QR код для быстрого подключения (опционально)

5. Сохранения (/backups):
   - Toggle "Автосохранение при выходе всех игроков"
   - Toggle "Автосохранение при остановке сервера"
   - Интервал автосохранения (select): 5, 10, 15, 30 минут
   - Кнопка "Сохранить мир сейчас"
   - Список сохранений:
     * Дата и время
     * Размер файла
     * Кнопка "Скачать"
     * Кнопка "Восстановить"
   - Настройка максимального количества сохранений
   - Toggle "Автоматическое удаление старых сохранений"

ТЕХНИЧЕСКИЙ СТЕК:
- React 18+ с TypeScript
- Tailwind CSS или shadcn/ui для UI
- Zustand для state management
- React Query для API запросов
- Axios для HTTP
- React Hook Form + Zod для форм и валидации

ИНТЕГРАЦИЯ С RAILWAY API:
- Базовый URL: https://api.railway.app/v1
- Аутентификация: Bearer token (Railway API Key)
- Endpoints:
  * GET /services/{serviceId} - статус сервера
  * GET /services/{serviceId}/variables - переменные окружения
  * PATCH /variables/{variableId} - обновление переменной
  * GET /deployments/{deploymentId}/logs - логи
  * POST /deployments/{deploymentId}/restart - перезапуск
  * GET /services/{serviceId}/domains - TCP Proxy адрес

ПЕРЕМЕННЫЕ ОКРУЖЕНИЯ RAILWAY:
- CUSTOM_SERVER_URL - URL кастомной сборки
- WORLD_SEED - seed мира
- GAMEMODE - режим игры
- DIFFICULTY - сложность
- PVP_ENABLED - PvP
- MAX_PLAYERS - максимальное количество игроков
- AUTO_SAVE_ENABLED - автосохранение
- AUTO_SAVE_INTERVAL - интервал автосохранения

ДИЗАЙН:
- Темная тема в стиле Minecraft (зеленые акценты)
- Современные карточки и компоненты
- Адаптивный дизайн
- Toast уведомления для всех действий
- Индикаторы загрузки
- Плавные анимации
